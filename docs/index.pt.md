Bem-vindo à Documentação do Vapor! Vapor é um framework web para Swift, permitindo que você escreva backends, APIs de aplicações web e servidores HTTP em Swift. O Vapor é escrito em Swift, que é uma linguagem moderna, poderosa e segura, oferecendo diversos benefícios em relação às linguagens de servidor mais tradicionais.

## Primeiros Passos

Se esta é a sua primeira vez usando o Vapor, vá até [Instalação → macOS](install/macos.md) para instalar o Swift e o Vapor.

Depois de instalar o Vapor, confira [Primeiros Passos → Olá, mundo](getting-started/hello-world.md) para criar sua primeira aplicação Vapor!
