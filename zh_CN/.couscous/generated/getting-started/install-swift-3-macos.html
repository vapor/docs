Install Swift 3: macOS

翻译：@孟祥月_iOS

在macOS上安装 Swift 3，你仅需要安装 Xcode 8 就可以了。

Install Xcode

从 Mac App Store 安装 Xcode 8 。

Open Xcode

Xcode 8 下载完成之后，你必须打开它，以完成安装。这将花费一段时间。

Check

通过如下命令再次检测是否安装成功：

curl -sL check.vapor.sh | bash

Toolbox

现在你可以继续 Install Toolbox。

Swift.org

如果你想了解更详细的安装 Swift 3.0 的指令，请查看 Swift.org 的更多指南。
